# 変更履歴

[English](CHANGELOG.md) | 日本語

pocket-claude の主要な変更をここに記録します。

## [v2.0.0] - 2026-05-09

### 追加
- **リクエストボディサイズ制限** - 設定画面から変更可能（MB 単位；0 = 無制限）；サーバー再起動なしで即時反映；`config.json` に保存
- **モデル一覧 API** (`GET /api/models`) - `config.json` からモデルリストを返す；取得失敗時は内蔵フォールバックリストを使用
- **画像添付** - プロンプトに画像を添付（base64、複数枚対応）
- **ステータスアイコンへの停止機能統合** - ヘッダーのステータスアイコンに停止ボタンを統合
- **クリップボードボタン** - ヘッダーにクイックコピーボタンを追加

### 修正
- `turning` フラグ導入によりアイドル状態の誤検知を抑制
- SSE 再接続時に `isRunning=true` が誤送出される問題を修正
- 予約投稿カードのプロンプト表示が途中で切れる問題を修正
- SSE 再接続後に実行中ステータスが正しく反映されない問題を修正
- ステータスアイコンのサイズ・アニメーション・見た目を修正
- ブラウザリロード後の履歴再生で ctx% が表示されない問題を修正
- スケジュールモーダルの入力欄で iOS Safari が自動ズームする問題を修正
- 常駐プロセスモードで自動再開・予約投稿・ステータスアイコンが機能しない問題を修正

### 内部変更
- `getClaudeSessionId()` を 3 ファイルの重複実装から `services/sessions.js` に集約

## [2026-03-28]

### 追加
- **予約投稿** - 指定日時にプロンプトを自動送信；出力パネルから編集・キャンセル可能；`scheduled-posts.json` に永続化（サーバー再起動後も復元）
- **言語切り替え** - 設定画面から日本語 / 英語 を切り替え可能；localStorage に永続化
- **レート制限自動再開** - レート制限発生時にパネルを表示してサーバーサイド自動再開を有効化；スケジュールを `schedules.json` に永続化
- **AskUserQuestion 表示改善** - ツール結果を自動展開して質問・選択肢をリスト表示
- **コードフォントサイズ設定** - 出力フォントサイズとは独立して設定可能
- 新規タブ作成時に現在のタブのモデル・Effort・Thinking 設定を継承
- Thinking コントロールをチェックボックスから 3 ステートセグメント（指定なし / オン / オフ）に変更

### 変更
- ツール展開表示を VS Code 拡張スタイルに刷新：Bash は IN/OUT 形式、Edit は赤/緑の diff 形式
- レート制限メッセージを赤字で表示；リセット時刻を目立つ表示に変更
- stderr の使用制限メッセージを `[err]` プレフィックスなしで `line-error`（赤）で直接表示
- モデルセレクタを左揃えに変更；Send ボタンの高さが textarea に連動
- ヘッダーボタンの高さを 30px に統一
- 設定 UI の Effort・Thinking ラベルを日本語化

### 修正
- CLI コマンドのパースエラー（`is_error: true`）が無視されていた問題を修正；赤字で表示
- HTTP 環境向け `crypto.randomUUID` ポリフィルを追加
- `parseResetTime` が `am/pm + IANA タイムゾーン` 形式に対応
- 予約リスト表示と履歴読み込みの競合（レースコンディション）を修正
- プロジェクト切り替え後、設定を閉じずに選択状態を即時更新するよう修正

## [2026-03-21]

### 追加
- **履歴ブラウザ** - 過去のセッションを閲覧・再開
- **SSE 出力バッファリング** - 再接続時に出力を復元
- **複数テーマ** - Blue Dark / Purple Dark（CSS 変数 + themes.js で管理）
- **コンテキスト使用量バー** - リアルタイムでトークン % を表示
- **プロジェクト管理** - ブラウザ UI からプロジェクトディレクトリを追加・削除；`ADDITIONAL_ALLOWED_DIRS` 環境変数サポート
- **グレースフルシャットダウン** - SIGTERM 対応
- **起動時自動修復** - 前回クラッシュ時の未完了ログエントリを自動でクローズ

### 変更
- タブ管理を localStorage のみに変更（サーバー側の tabs.json 廃止）
- セッション永続化を分離：pocket-session ID と Claude session ID を独立管理
